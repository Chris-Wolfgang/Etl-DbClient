Remarks

The target type MUST be declared partial so the generator can add members to it. Every public property maps to a column of the same name unless overridden with DbColumnAttribute.

First v0.5.0 cut: Type-level analysis only — the generator emits a public const string Insert and a static void Bind(Dapper.DynamicParameters parameters, T record) helper. Future versions will expand to Update, Delete, and Select with proper integration into DbExtractor / DbLoader.
