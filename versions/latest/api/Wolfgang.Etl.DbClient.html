Wolfgang.Etl.DbClient

Namespace Wolfgang.Etl.DbClient

Classes

DbClientOptions

Process-wide knobs for Wolfgang.Etl.DbClient. Settings here are read from the global Dapper type-map delegate that ColumnAttributeTypeMapper installs, so they affect read-side column resolution only.

DbColumnAttribute

Optional per-property override for a DbTableAttribute-decorated record. Maps the annotated property to a column whose name differs from the property name, or marks it as Skipped from the generated SQL.

DbExtractor<TRecord>

Extracts records from a database query as an asynchronous stream. Uses Dapper for column-to-property mapping, supporting [Column] attribute and convention-based name matching.

DbLoader<TRecord>

Loads records into a database via INSERT or UPDATE commands. Uses Dapper for parameter mapping from POCO properties.

DbReport

Progress report for ADO.NET extraction and loading operations. Extends the base Wolfgang.Etl.Abstractions.Report with database-specific context.

DbTableAttribute

Marks a record / class as mapping to a database table. Triggers the Wolfgang.Etl.DbClient source generator to emit a partial class with reflection-free Insert SQL + parameter-binding helpers.

RowFailedEventArgs<TRecord>

Payload for DbLoader{TRecord}.RowFailed. Fires once per row that failed to load when ErrorHandling is Skip.
