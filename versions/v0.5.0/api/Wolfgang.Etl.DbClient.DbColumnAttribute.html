Optional per-property override for a DbTableAttribute-decorated record. Maps the annotated property to a column whose name differs from the property name, or marks it as Skipped from the generated SQL.

[AttributeUsage(AttributeTargets.Property, Inherited = false, AllowMultiple = false)]
